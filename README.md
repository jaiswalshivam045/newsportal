# News Portal

A PHP and MySQL based News Portal web application that allows users to browse news articles, create accounts, bookmark articles, and manage profiles.

## Features

* User Registration and Login
* User Profile Management
* News Article Display
* Bookmark Favorite Articles
* Responsive User Interface
* MySQL Database Integration
* Secure Authentication System

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/jaiswalshivam045/newsportal.git
```

2. Create a MySQL database.

3. Import the database:

```sql
news_portal.sql
```

4. Configure database credentials in the configuration file:

```php
$db_host = "localhost";
$db_name = "your_database_name";
$db_user = "your_username";
$db_pass = "your_password";
```

5. Upload the project files to your web server's `htdocs` or `public_html` directory.

6. Open the website in your browser.
